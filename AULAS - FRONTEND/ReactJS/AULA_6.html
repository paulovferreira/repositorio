FLEXBOX - Continuação

POSICIONAMENTO DOS ITENS

O posicionamento dos itens dependerá de algumas condições:
1) Qual a largura dos itens internos da minha DIV Conteiner;
2) Qual a FLEX-DIRECTION do meu Conteiner
Observação. Se utilizado o dimencionamento pelo atributo FLEX, o css não conseguirá posicionar os itens desse maneira.

JUSTIFY-CONTENT

Alinha os itens de acordo com o Direction escolhido
Se for Linha, organiza os itens internos da DIV na linha (horizontalmente).
e for Coluna, organiza os itens internos da DIV na Coluna (Verficalmente.)
Possibilidades: Center, flex-start, Flex-end, Left, Right,
Space Arround (espaçamento igual)
Space-Between (Espaço igual exceto o primeiro e o ultimo);

ALIGN-ITEMS

Alinha os itens no sentido oposto do JUSTIFY-CONTENT.


ALINHAMENTO DE ITENS INDIVIDUALMENTE

Para alinhar itens dentro de uma DIV de maneira individual, basta utilizar a propriedade:
ALIGN-SELF: [Meu Alinhamento]


DETERMINAÇÃO DA ORDEM DE EXIBIÇÃO.

A propriedade ORDER: dita a ordem que os itens serão exibidos.
Se não houver especificação, o valor atribuido será 0, tornando-o prioridade.

FLEX WRAP

Determina como será feita a quebra de linha entre os componentes a medida que eles extrapolam o WIDHT máximo da tela
OBS. Só funcionam caso o tamanho dos itens esteja determinado de maneira FIXA.

flex-wrap : wrap;

Ao utilizar o flex-wrap, podemos especificar o alinhamento dos itens de acordo com
o contexto onde ele está inserido.
OBS. Só funciona para alinhamento de conteúdo, quando houver quebra de linha por meio do wrap.

align-content: [alinhamento]
