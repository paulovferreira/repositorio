REQUISIÇÕES

Requisições são a base de todo sistema integrado. É por meio delas que podemos integrar nossos sistema a sistemas externos e sobretudo, integrar nosso FrontEnd ao nosso BackEnd.
Existem muitas maneiras de realizar requisições em React, vamos iniciar pela mais básica. Requisições GET puras.

REQUISIÇÕES GET

Uma requisição GET, tem o objetivo de solicitar a uma API externa os dados provenientes dessa API para listar suas informações em ambiente interno. Para isso, vamos construir nosso exemplo.

Criando Componentes para exibição de requisições.
Consulte o arquivo requisicoes.tsx do Aplicativo exemplo.

Utilizando FETCH

FETCH é um comando nativo do React para conexão com APIS externas. Iremos explanar seu uso no aplicativo exemplo.

No exemplo utilizaremos uma API pública que contém produtos para serem exibidos.
LINK: "https://fakestoreapi.com/products/"

Utilizando FETCH com TYPES separados

É muito importante que tenhamos os Types das nossas listas criados separadamente.
Desta maneira, é possível organizar o resultado da conexão com sua API de modo que não se perca a nome original do seu resultado.

IMPORTANTE
O seu arquivo Type que ordenará o resultado da API deverá ter EXATAMENTE o mesmo nome dos elementos do seu Json resultado.
