RENDERIZANDO CONTEÚDO CONDICIONAL

Por vezes será necessário exibir ou não determinado componente ou divisão na tela a depender de uma condição pré estabelecida.
É o caso classico de elementos de "Aguarde" exibidos enquanto dados são buscados na base de dados.

criando a constante para armazenar o estado do componente.
cont[show, setShow] = useState(false);

Após criar a constante, será necessário criar um botão para executar a modificação da constante.
< button> MOSTRAR < /button>

Logo após o botão, vou criar a DIV que irá ser exibida ou não, a partir da verificação da minha constante.

< div>
ESTE TEXTO SÓ SERÁ EXIBIDO SE O SHOW FOR = A TRUE.
< /div>

Diante disso, posso criar um evento no meu botão para alterar o estado da minha constante SHOW

Criando a função para o evento
function handleClickMostrar (){
setShow(true);
}

Chamando a função criada no evento Click:
< button onclick="handleClickMostrar"> Mostrar < /button>

Posso ainda tornar o meu botão dinamico, alterando o status visual do meu elemento atravez do click do botão.


if(show){
setShow(false)
} else {
setShow(true)
}

Podemos ainda trabalhar a alteração de vários componentes simultâneamente mediante a alteração da mesma variável.

Para isso, utilizamos código JS puro
< button onclick="handleClickMostrar"> { show ? "Ocultar" : "Mostrar" } < /button>

Dessa maneira, nossos elementos são exibidos / ocultados e o texto do botão também é alterado.
