RENDERIZAÇÃO DE LISTAS

Por diversas vezes, será necessário renderizar listas para exibição dinâmica na tela.
As listas podem ser criadas dinamicamente ou vir atravez de uma fonte de dados, porém, independente de sua origem, a exibição se dará de maneira automática.

Criando uma lista.

let list = [
'Nome 1',
'Nome 2',
'Nome 3',
'Nome 4'
]

Com a lista criada, precisamos RENDERIZÁ-LA
< h2>Lista de Nomes< /h2>
< ul>
{list.map((item)=>(
< li>{item}< /li>
))}
< /ul>

Isso fará com que todos os itens da minha lista sejam exibidos de maneira dinamica num LOOP.

RENDERIZAÇÃO ÚNICA DE LISTAS

Se olharmos no "Inspecionar Elementos" do navegador, em Console, vamos ver que mesmo que a lista seja exibida, o navegador exibe um alerta.
Neste alerta, é informado que o item da minha lista não possui uma CHAVE ou KEY, oque torna meu item "anonimo", sem identificação.
Isso se tornará um problema no momento em que precisarmos identificar qual item de uma lista estamos utilizando em uma operação, por exemplo.

Criando uma chave única para o item.
< ul>
{list.map((item)=>(
< li key="1" >{item}< /li>
))}
< /ul>

Perceba que a mensagem exige que seja informado uma Key, porém, se informarmos um valor fixo, esse valor será replicado para todos os itens da lista, logo, o problema não é resolvido.
Para isso, precisamos utilizar o parâmetro INDEX, presente no MAP.

< ul>
{list.map((item, index)=>(
< li key={index} >{item}< /li>
))}
< /ul>
Dessa meneira, o item passa a ter uma chave única.

RENDERIZANDO LISTAS DE OBJETOS

O processo de renderização de uma lista de objetos se dá da mesma forma. A única diferença é que em nossa criação, criaremos a lista especificando todos os valores do objeto.

let list = [
{name: 'Nome 1', age: '30' },
{name: 'Nome 2', age: '35' },
{name: 'Nome 3', age: '40' },
{name: 'Nome 4', age: '45' },
{name: 'Nome 5', age: '60' },
]

Para utilizar esse objeto que está vindo por meio do meu loop, basta acessar as propriedades do item que criamos em sua declaração
< ul>
{list.map((item)=>(
< li key="1" > Nome: {item.name} - {item.age} Anos< /li>
))}
< /ul>


Renderizando Listas de componentes

Nem sempre a origem da lista estará no mesmo local onde ela será exibida. Nestes casos, tornamos os itens da lista COMPONENTIZADOS, para exibição.

Criando o componente ItemLista
type Props = {
dados: {
name: string;
age: string;
}
}
function ItemLista ({dados}: Props) {
return(
< li>Nome: {dados.name} - Idade: {dados.age}< /li>
)
}
export default ItemLista;

Uma vez criado, utilizaremos a PROP do componente para passar o objeto a ser exibido.

Criando a lista por meio de componentes
É preciso importar o componente ItemLista

< ul> {listObj.map((item, index)=>( < ItemLista key={index} dados={item} /> ))} < /ul>
