CRIAÇÃO DOS COMPONENTES

Uma vez criado nossos cabeçalhos e rodapés das páginas, podemos criar links de referencia para páginas já existentes ou para nossas nossas novas páginas.

CRIANDO TABELAS PARA ORGANIZAÇÃO DO CABEÇALHO.

Neste primeito momento, vamos criar uma tabela para ciar um menu no cabeçalho da nossa página.
Para isso,vamos utilizar a estrutura já conhecida abaixo:

< table> < tr> < td>< /td> < /tr> < /table>

O Menu ficará parecido com o menu abaixo:
menu 1 menu 2 menu 3 menu 4 menu 5 menu 6

UTILIZANDO VARIÁVEIS PARA NOMEAÇÃO DO MENU

Para facilitar e tornar o código mais estruturado, não utilizaremos os nomes dos menus de maneira explicita no código.
Com JAVASCRIP é possível referenciar essa informação de maneira VARIÁVEL para que se torne dinamica.

Para isso, utlizamos o JAVASCRIP PURO declarando as variáveis essa forma:

let [variavel]: string = "[valor]";

Para utilizar a variável criada, basta referenciá-la assim:

Olá, o conteúdo da minha variável é: {variavel}
