FLEXBOX

Caixas Flexiveis são utilizadas para montagem do Layout de páginas HTML de maneira responsiva e auto organizável.
Com caixas flexíveis é possivel prever e determinar que os elementos jamais sairão da posição onde foram construídos de maneira desordenada.

Todo elemento flexível, parte de um container, ou seja, uma DIV.
Quem determinará que o conteiner criado será flexível sempre será o CSS.

NO HTML
< DIV ClassName="container">< /DIV>

NO CSS
.container{
display: flex;
}


A partir da criação do seu conteiner principal, a criaçao dos itens internos do conteiner passarão a respeitar o elemento PAI.


FLEX-DIRECTION

A propriedade flex flex-direction, utilizada no seu conteiner principal, ditará como os elementos deverão se comportar dentro do elemento conteiner principal.

ROW
Os elementos se ordenarão dentro da mesma Linhas.
COLUMN
Os elementos se ordenarão dentro da mesma Coluna.

DIV -> DIV

A organização dos elementos dentro de uma Div flexível passa a ser realizada por outras DIVs sempre que necessário. Tornando assim, a construção do layout do site muito mais ordenada e controlável.

NO JSX
function cabecalho(){
return(
< div className='container_cabecalho'>
< div className='divclass1' teste < /div>
< div className='divclass2' >
< Menu />
< /div>
< div className='divclass3'> Logout < /div>
< /div>
)
}



NO CSS
.container_cabecalho{
display: flex;
flex-direction: row ;
background-color: gray;
}

.divclass1, .divclass2, .divclass3 {
margin: 10px;
padding: 10px;
}

.divclass1{
flex:1;
background-color: red;
}

.divclass2{
flex:5;
background-color: green;
}

.divclass3{
flex:1;
background-color:red;
}

DIVS INTERNAS - definição de largura

Posso definir as larguras das minhas DIVs dentro de Conteiners de muitas maneiras, de acordo com a necessidade da aplicação, porém, existem duas maneiras mais tradicionais:

Com o convencional WIDTH
Dessa maneira, a largura do seu componente se tornará FIXA de acordo com o especificado
WIDTH: 100px;

UTILIZANDO FLEX
O FLEX permite que o tamanho da sua DIV interna seja variável de acordo com o escopo onde está inserida, ou seja, de acordo com o tamanho do seu pai.

A especificação do valor da propriedade FLEX é:
1 = 100% do tamanho.
A medida que o valor de FLEX aumenta, ele divide proporcionalmente os elementos de modo que se encaixem perfeitamente.
