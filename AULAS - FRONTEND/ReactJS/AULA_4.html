COMPONENTIZAÇÃO DO CABEÇALHO E RODAPÉ DA PÁGINA E SEUS ELEMENTOS.

Não é interessante que todos os componentes da tela estejam reunidos num mesmo arquivos. Para tornar o react realmente aproveitável na plenitude de suas funções, criaremos componentes para serem trabalhados de maneira individual.

Para isso criaremos a seguinte estrutura:

src
-- Components
---- Cabecalho
---- Rodape
---- Menu
---- [Seu Projeto]

Ainda criaremos um arquivo único para a estilização dos componentes via CSS.

src
-- Estilo

PRÁTICA

Na aula de hoje, componentizaremos todos os elementos criados no App.tsx de maneira com que fiquem 100% independentes.
