PROPS EM JSX

Muitas vezes é necessário utilizar valores variáveis de ma tela para outra, como por exemplo, a mudança do texto do meu cabeçalho a depender do local de onde ele é chamado.
Para isso, existem as PROPs.
Num comparativo simplorio com o HTML:

Em HTML
< a href="" > Clique aqui < /a>
Onde:
A = Elemento
href = Atributo

EM JSX
< Cabecalho titulo="Minha Página Principal" />
Onde:
Cabecalho = Componente
titulo = PROP

PREPARANDO O COMPONENTE PARA RECEBER UMA PROP

Para que possamos utilizar valores variáveis em componentes, precisamos prepará-lo para receber a variável.

type Props = {
titulo: String;
}
function cabecalho(props: Props ){
return(
< div className='cabecalho'> Cabecalho da página
)
}
export default cabecalho

Após declarar a PROP e torná-la disponível para utilização, podemos manipulá-la dentro da minha página.

Essa é a propriedade que recebi de quem me chamou: {props.titulo}

Desobrigando o uso de uma prop criada

Ao criar uma PROP, percebemos que ela automaticamente passa a ser obrigatória.
Para desobrigar a utilização da prop, basta declará-la em seu TYPE com o caracter "?" após o seu nome, assim:

type Props = {
titulo?: String;
}

Após declarar seu componente para receber a PROP será necessário informar no momento da invocação do Componente.
Para isso, fazemos assim:

< Cabecalho titulo="Este é o meu título" />

PRÁTICA

Criar uma PROP em um de seus componentes, para exibir uma informação importante para a página seguinte.
