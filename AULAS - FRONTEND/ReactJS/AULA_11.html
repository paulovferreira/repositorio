EVENTOS EM JSX.

Por vezes, temos a necessidade de criar eventos para determinadas ações dentro da nossa págoina. Sempre que houver essa necessidade, utilizamos um EVENTO.
Os eventos são criados para tornar os elementos intetivos entre si.

Criando o evendo click de um botão:
Criação do Botão:
< button onclick={}> CLIQUE AQUI < /button>


Perceba que agora temos um botão esperando uma ação mendiante a um click.
O evento Onclick é uma PROP do próprio botão que conectará uma ação criada pelo próprio desenvolvedor.

Criando a função do click do botão.
function handleButtonClick () {
Alert("O botão foi clicado.");
}

Conectando o evento criado ao botão:
< button onclick={handleButtonClick}> CLIQUE AQUI < /button>

Assim, ao clicar no botão, a mensagem de alerta será exebida.


STATE EM COMPONENTES

O state de un componente está relacionado ao seu ciclo de vida.

Ciclo de vida natural do componente
É CRIADO -> MODIFICADO -> DESTRUÍDO.

APÓS a sua criação, ele pode ser modificado em "TEMPO DE EXECUÇÃO" e isso modifica o seu estado original.

EXEMPLO
let numero: number = 10;
...
return (
< div>
O numero a ser exibido é: {numero}
< /div>
)

Para modificar o ESTADO atual do meu componente, basta eu modificalo, via EVENTO. Assim:

function handleButtonClick () {
numero += 10;
}

Perceba que ao fazer dessa maneira, O ESTADO ATUAL do meu componente não é modificado, pois a linha de declaração da minha variável tem seu valor fixo atribuído.
Dessa forma, ao modificar o ESTADO de um componente, toda a sua estrutura é recriada, fazendo assim, com que ele volte ao seu ESTADO original.

Alterando o STATE de um Componente.

Assim sendo, não podemos simplesmente atribuir um valor a variável e esperar que ela seja RENDERIZADA automaticamente. Precisamos fazer isso, via FUNÇÃO.

IMPORTANDO A BIBLIOTECA DE STATES
import {UseState} from 'react';

Criando constantes para a mudança controlada de valores: const [numero, setNumero] = UseState(0);
const [nome, setNome] = UseState(0);

Alterando o valor da minha constante via função:
...
setNumero(10);
setNome('Paulo');

Dessa forma, nosso componente é modificado apenas em seus atributos que sofreram alteração e não no componente todo.
