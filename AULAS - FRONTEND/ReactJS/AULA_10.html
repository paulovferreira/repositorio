Router - Rotas Dinamicas

Por muitas vezes, precisamos criar rotas dinamicamente, em casos onde existam muitas páginas filhas.
Afim de criar essas fotas de maneira automática, rezamos a criação em massa, dessa forma:

< Route path="/sobrealuno/:parametro" element={ < SobreItemAluno/ >} />


Para isso, precisaremos CRIAR a página SobreItemAluno para tornar seu direcionamento dinâmico.


Isso fará com que o ROUTER identifique dinamicamente o parâmetro enviado e processe de acordo com uma ação específica do próprio usuário;

Ao acessar QUALQUER link após o subdominio "sobrealuno", o sistema direcionará sempre ao mesmo elemento, cabendo assim o tratamento de cada rota por parte do usuário segundo o parâmetro enviado.

IDENTIFICANDO O PARÂMETRO RECEBIDO

Após preparar o Router para enviar esse parâmentro, nosso elemento SobreItemAluno precisa estar preparado para processar esse parâmetro.

Importando a biblioteca PARAM
import {useParams} from 'react-router-dom';

Utilizando o parâmetro enviado:
Declaramos uma constante para armazenar esses paramentros.

const params = useParams();

A partir de agora, todos os paramentros enviados estão dentro da constante: "PARAMS";

Utilizando os parâmentro enviados:
Essa é uma página sobre o aluno: {params.paramentro}

"parametro" = nome do paramentro enviado.

Redirecionamento de Páginas

Por vezes, será necessário navegar entre páginas do mesmo projeto ou ainda voltar para páginas anteriores e fazer isso, por meio de links (âncoras) nunca é a melhor opção.
Para isso, utilizamos o NAVIGATE.

Importanto a biblioteca do Navigate
import {useNavigate} from 'react-router-dom';

Criando a constante para utilização do navigate:
const navigate = useNavigate();

Agora podemos utilizar todos os recursos do navigate em nossa tela.

Criando botão VOLTAR

Para criar o botão voltar, faremos o uso da constante NAVIGATE.

Criando o botão voltar:
< button onClick={handleVoltar}> Voltar < /button>

Criando a função handleVoltar
function handleVoltar(){
navigate(-1);
}


Dessa maneira, podemos criar botões para navegar em qualquer ação dentro do nosso JSX.

Criando botões para rotas específicas:

Criando o botão de navegação
< button onClick={handleHome}> Voltar para Home < /button>

Criando a função handleHome
function handleVoltar(){
navigate('/');
}
Neste formato, podemos utilizar o botão para direcionar para qualquer link, bastando informar a rota criada no seu arquivo padrão de rotas.
