MODIFICAÇÃO DO STATE DE INPUTS

Inputs são componentes para inserção de textos que servem para diversas finalidades e principalmente para criação de cadastros.
A alteração do seu ESTADO se dá de maneira diferente dos demais componentes.
Sua altearção se dará por meio do evento onChange que capta a modificação em tempo real.

IMPORTANDO BIBLIOTECAS
import React from 'react'.
import {UseState} from 'react';

Criando constantes para os valores do input
const [name, SetName] = useState('');

Criando função para alteração do valor da constante
function handleInput (event: React.ChangeEvent) {
SetName(event.target.value);
}

Criando o retorno visual do componente.
return(
< div>
Nome:
< input type="text" value={name} onChange={handleInput} />
< hr>
Seu nome é: {handleInput}
< /div>
)
