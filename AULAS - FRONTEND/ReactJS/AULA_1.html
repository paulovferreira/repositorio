AULA 1 - REACT JS

É uma biblioteca criada em JS criada pelo FACEBOOK. Em países mais precários, a tendencia é que os computadores sejam mais simples.

o Facebook acabou ficando mto pesado pela sua interface
Pensando nisso, o time de desenvolvimento criou essa biblioteca para rodar mais rápido. Então, os elementos foram retrabalhados para serem exibidos mais rapidamente.

A cada carregamento da tela, o navegador precisa construir os elementos para exibição o que demanda muito processamento. Exemplo: se na tela cabem 5 posts, ao rolar, o número de postagens ia aumentando oq acabava travando o navegador.

O react trabalha com o carregamento DINAMICO / VIRTUAL dessas informações.
O react espelha o conteúdo da página de maneira virtual o conteúdo exibido na página. - Ao atualizar, o react compara os arquivos (VIRTUAL E TELA) e aplica somente a mudança na tela, sem a necessidade de reprocessar todo o conteúdo da pagina

SPA - SINGLE PAGE APPLICATION

Aplicação de uma página.

Esse conceito garante que tudo que está sendo visualizado na tela, estará presente na mesma página.

A aparencia pode até ser multipáginas, mas a virtualização garante que tudo não passa e de uma única página.

INSTALANDO O NODE JS

- Acesse nodejs.org;
- Baixar a versão desejada.
- Instalar normalmente.

PARA TESTAR
No TERMINAL / PROMPT, executar o comando:

node -v ou node --version ou npm --version

CRIANDO MEU PRIMEIRO PROJETO - UTILIZANDO CRA

utilizando a técnica CRA - Create React Application
No PROMPT navegue a tela para criação do projeto;

npx create-react-app [nome_do_projeto] --template typescript

(sempre minúsculo)
Com o projeto criado, basta acessar a pasta por meio do vsCode. No terminal, executamos o comando:

npm start

O projeto criado, deverá ser exibido.

CRIANDO MEU PRIMEIRO PROJETO - UTILIZANDO VITE

O Vite é um creador de projetos que auxilia no momento da organização do projeto.
Além disso, cria projetos integrados entre diferentes plataformas.

Para criar projetos utilizando vite
No prompt de comandos:

npm init vite react-vite -- --template react-ts

Obs. A partir da versão 7 da npm, utiliza-se 4 "-".

Após a criação, é preciso instalar os pacotes. Estando posicionado na pasta onde está o projeto no prompt executar:

npm install

Para executar o projeto:

npm run dev

Pronto, seu projeto está no ar.
