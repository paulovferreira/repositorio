PARAMETROS NA CHAMADA DA API.

Passar parâmetros para a chamada de uma API, é muitas vezes um pré requisito fundamental para obter um resultado esperado.
Para isso, precisamos ajustar nossa classe geral de APIs para tratar o parametro de maneira eficiente e assim, concatenar seu conteúdo dentro da URL da nossa API.

Os exemplos e comentários a nível de código estão presentes no arquivo:
requisicoesParam.tsx do projeto exemplo.
